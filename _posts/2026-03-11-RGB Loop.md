The purpose of this assignment was to make a while loop that only printed text every tenth time. In the RGB loop, each color had to be printed in triplets. (3) Red, then (3) Green, then (3) Blue.

One tip I would give myself for the fure is it to think outside of the nested commands. I found myself trying to come up with some arithmatic solution for 0-2, 3-5, and 6-8, when a simple reset command was all I needed.
